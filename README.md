# Bank-Management-System

A simple **console-based Bank Management System** built using **C language**.  
This project is designed to demonstrate core programming concepts like **file handling, structures, functions, and user authentication** through a real-world banking example.

---

## Overview

This system allows users to create and manage bank accounts, perform transactions, and view account details.  
An **Admin panel** is also included to manage all customer accounts and maintain records securely.

The project is completely file-based, so no external database is required.

---

## Key Features

### Customer Side
- Create a new bank account  
- Login using Account Number and PIN  
- Deposit money  
- Withdraw money  
- Check account balance  
- View mini statement (transaction history)  
- Change account PIN  
- Calculate simple interest  
- Transfer funds to another account  

### Admin Side
- Secure admin login  
- View all customer accounts  
- Search account by account number  
- Update account details  
- Delete account  
- Freeze / unfreeze accounts  
- Unlock locked accounts  
- View bank statistics  
- Backup account and transaction data  
- Export account data to Excel (CSV format)  

---

## Technologies & Concepts Used

- C Programming  
- Structures  
- File Handling (`.txt` and `.csv` files)  
- Functions & Modular Programming  
- Date & Time Handling  
- Input Validation
